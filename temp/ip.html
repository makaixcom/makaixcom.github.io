CentOS 8系统，弃用了network.service ，采用NetworkManager（NM）为网卡启用命令，当然CentOS 8 也可以安装network.service作为网卡服务，但是为了适应版本，我们在这里还是得来看下nmcli的命令的使用。 网卡停止和启动命令： 启动 nmcli connection up ens33 停止 nmcli connection down ens33

重启命令： [root@localhost ~]# nmcli connection reload

查看网卡设备详细信息 nmcli device show / nmcli device show eth0

查看网卡设备状态 nmcli device status

查看网卡信息 nmcli c / nmcli connection show

查看所有活动连接 [root@localhost ~]# nmcli c show --active

# 更换IP

修改/etc/sysconfig/network-scripts/ifcfg-ens33

# 修改以下内容
ONBOOT=yes
BOOTPROTO=static

# 追加以下内容
IPADDR=192.168.3.129
NETMASK=255.255.255.0
GATEWAY=192.168.3.2
DNS1=223.6.6.6
DNS2=180.76.76.76
