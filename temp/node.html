# node安装

官网下载http://nodejs.cn/download/

一路next（只需要更改路径）

解决报错

npm.cmd中prefix -g 替换为prefix --location=global

设置淘宝源

npm config set registry https://registry.npmmirror.com

验证

npm config get registry

安装yarn

npm install -g yarn
