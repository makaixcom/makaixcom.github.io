# idea的目录解析

bin：容器，执行文件和启动参数等
help：快捷键文档和其他帮助文档
jbr：jetbrains runtime ，其实就是jb公司自己定制的jdk，供idea启动用的
lib：idea 依赖的类库
license：各个插件许可
plugin：插件

以下两个目录，只有idea运行过一次才会有。

# 系统、配置、日志和插件目录

默认位置是在 C:\Users\用户名\AppData\Roaming\JetBrains\IntelliJIdea2021.2 下。可以通过在安装目录/bin/idea.properties文件中的配置项更改

# idea.config.path=${user.home}/.IntelliJIdea/config
idea.config.path=D:/dev/soft/jetbrains/ideaconfig/config

# 缓存目录

缓存默认在C:\Users\makai\AppData\Local\JetBrains\IntelliJIdea2021.2下，通常这个目录可以通过在安装目录/bin/idea.properties文件中的配置项更改

# idea.system.path=${user.home}/.IntelliJIdea/system
idea.system.path=D:/dev/soft/jetbrains/ideaconfig/system
