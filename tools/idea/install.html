# 前言

原本安装的时候，是需要下载 IntelliJ IDEA 的，但是实际感觉，IntelliJ Toolbox 更好用一些，所以下载的是 Toolbox的安装方式。

# Toolbox 下载

在 https://www.jetbrains.com/products/open in new window 页画找到 toolbox app 对应的框框，点击下载

# Toolbox 配置

打开设置窗口，点击右上角的六边形，打开 Settings，调整以下设置：

  • language 改为 简体中文
  • 主题改为 深色调
  • 在系统启动时 启动Toolbox App打勾
  • 自动更新 Toolbox App打钩
  • 工具安装位置 ，一定要更改。我的 D:\dev\soft\JetBrains\Toolbox

# 下载 IntelliJ IDEA

直接点击下载就可以了。
